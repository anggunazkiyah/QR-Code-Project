# QR Code Generator

A simple and efficient QR code generator built with Node.js that allows you to create QR codes for text, URLs, and other data.

## Features

- Generate QR codes for any text or URL
- Clean and simple interface
- Fast and reliable QR code generation
- Lightweight and easy to use

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/anggunazkiyah/QR-Code-Generator.git
   cd QR-Code-Generator
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Usage

Run the application:
```bash
node index.js
```

Follow the prompts to enter the text or URL you want to convert into a QR code.

## How it Works

1. The application prompts you to enter text or a URL
2. It generates a QR code using the QR code library
3. The QR code is saved or displayed for use
